# StudyHub Roadmap

This roadmap documents the intended execution sequence for the PDF Learning System and future epics. Each sub-phase must pass through the strict cycle: **Build → UAT → Fix → Stabilize → Freeze**.

## [CURRENT] UX Stabilization Round 1
**Vision:** Halt introduction of new major features to focus on usability, design consistency, and core user experience across the platform.

- **[PENDING] Priority 1:** Onboarding Flow Redesign
- **[PENDING] Priority 2:** Category Colors System
- **[PENDING] Priority 3:** Category Daily Target Visualization
- **[PENDING] Priority 4:** Sprint Explanation & Guidance
- **[PENDING] Priority 5:** Mandatory Category Validation
- **[PENDING] Priority 6:** Dashboard Empty States
- **[PENDING] Priority 7:** Analytics Labels & Insights
- **[PENDING] Priority 8:** Knowledge Trainer Explanation
- **[PENDING] Priority 9:** Clock Settings UX
- **[PENDING] Priority 10:** Dashboard Layout Simplification

*Exit Criteria: Finish Firebase Storage setup, Freeze Phase 14, and resume Phase 15.*

## Phase 15: PDF Learning System
**Vision:** A PDF is not a file; it is a learning asset.

- **[PAUSED] Phase 15.1:** PDF Reader Foundation (In-app overlay, navigation, zoom, page persistence).
- **[PENDING] Phase 15.2:** Reading Progress & Analytics (Track total pages, reading %, time spent).
- **[PENDING] Phase 15.3:** PDF Notes Integration (Highlight text → generate contextual Note).
- **[PENDING] Phase 15.4:** PDF Question Integration (Highlight text → generate Question).
- **[PENDING] Phase 15.5:** PDF Flashcard Integration (Highlight text → generate dedicated Flashcard).
- **[PENDING] Phase 15.6:** PDF Knowledge Trainer Link (Funnel generated Questions/Flashcards to active review queues).
- **[PENDING] Phase 15.7:** Local Study Folder Integration (Connect a local folder via File System Access API, scan PDFs, sync metadata, and natively read files without cloud upload costs).

## Future Epics
- **PDF Collections:** Organizing related reading materials without flat-list fatigue.
- **Advanced Spaced Repetition:** Expanding the Sprint and Knowledge Trainer algorithms.
